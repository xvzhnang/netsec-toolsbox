# KaTeX 本地文件说明

## 文件位置

请将 KaTeX 0.16.9 的文件放置在此目录下：

### 必需文件

1. **JavaScript 文件**：
   - `katex.min.js` 或 `dist/katex.min.js`

2. **CSS 文件**：
   - `katex.min.css` 或 `dist/katex.min.css`

3. **字体文件**（如果使用 `dist` 目录结构）：
   - `dist/fonts/` 目录下的所有字体文件

## 下载方式

### 方式 1：从 npm 包提取

```bash
# 安装 KaTeX
npm install katex@0.16.9

# 复制文件到 public/katex/
# Windows PowerShell:
Copy-Item node_modules/katex/dist/* public/katex/dist/ -Recurse
Copy-Item node_modules/katex/katex.min.js public/katex/
Copy-Item node_modules/katex/katex.min.css public/katex/
```

### 方式 2：从 GitHub 下载

1. 访问：https://github.com/KaTeX/KaTeX/releases/tag/v0.16.9
2. 下载 `katex.tar.gz` 或 `katex.zip`
3. 解压后将 `dist` 目录内容复制到 `public/katex/dist/`

## 目录结构

推荐结构：

```
public/katex/
├── katex.min.js
├── katex.min.css
└── dist/
    ├── katex.min.js
    ├── katex.min.css
    └── fonts/
        ├── KaTeX_AMS-Regular.woff2
        ├── KaTeX_AMS-Regular.woff
        ├── KaTeX_AMS-Regular.ttf
        └── ... (其他字体文件)
```

## 注意事项

- 确保所有字体文件都在 `dist/fonts/` 目录下
- CSS 文件中的字体路径会自动指向 `dist/fonts/`
- 如果文件不存在，应用会尝试多个路径，但最终会失败并记录错误
