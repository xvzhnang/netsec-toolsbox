# 字体文件目录

请将 JetBrains Mono 字体文件放置在此目录下。

## 需要的字体文件

请将以下字体文件复制到此目录：

- `JetBrainsMono-Regular.woff2` (推荐，体积小)
- `JetBrainsMono-Regular.woff` (备选)
- `JetBrainsMono-Regular.ttf` (备选)
- `JetBrainsMono-Bold.woff2` (推荐)
- `JetBrainsMono-Bold.woff` (备选)
- `JetBrainsMono-Bold.ttf` (备选)
- `JetBrainsMono-Italic.woff2` (推荐)
- `JetBrainsMono-Italic.woff` (备选)
- `JetBrainsMono-Italic.ttf` (备选)
- `JetBrainsMono-BoldItalic.woff2` (推荐)
- `JetBrainsMono-BoldItalic.woff` (备选)
- `JetBrainsMono-BoldItalic.ttf` (备选)

## 字体下载

可以从以下地址下载 JetBrains Mono 字体：

- 官方网站：https://www.jetbrains.com/lp/mono/
- GitHub：https://github.com/JetBrains/JetBrainsMono

## 文件格式说明

- **woff2**：最推荐，压缩率最高，现代浏览器支持
- **woff**：备选，压缩率较高，兼容性好
- **ttf**：备选，未压缩，兼容性最好但体积较大

程序会按优先级自动选择可用的字体文件。
